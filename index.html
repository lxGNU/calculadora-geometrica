📌 Estructura básica de un script en Python

Todo script en Python comienza importando las librerías necesarias y definiendo funciones principales.

1. Importar módulos necesarios

import math  # Para operaciones matemáticas avanzadas (pi, potencias, etc.)

math nos da acceso a constantes como π y funciones matemáticas.

2. Crear la función principal

def calcular_figuras():
    while True:  # Bucle infinito para menú interactivo
        print("1. Opción 1")
        print("2. Opción 2")
        opcion = input("Seleccione: ")

3. Estructura de control (if/elif/else)

if opcion == "1":
    # Código para cuadrado
    lado = float(input("Ingrese lado: "))
    area = lado ** 2
    perimetro = 4 * lado
    print(f"Área: {area}")
elif opcion == "2":
    # Código para círculo
    radio = float(input("Ingrese radio: "))
    area = math.pi * radio ** 2

4. Manejo de errores (opcional pero recomendado)

try:
    lado = float(input("Ingrese lado: "))
    if lado <= 0:
        print("El valor debe ser positivo")
    else:
        # Continuar con cálculo
except ValueError:
    print("Error: debe ingresar un número")

5. Punto de entrada del script

if __name__ == "__main__":
    calcular_figuras()

Esta línea permite que el script sea ejecutable directamente o importable como módulo.

💡 Consejos Python

  • Usa f-strings para formatear texto: f"Resultado: {valor:.2f}"
  • math.pi es más preciso que usar 3.14159
  • ** para potencias, no ^ (ese es para XOR)

📌 Estructura básica de un script en Bash

Los scripts de Bash comienzan con shebang y pueden usar bc para cálculos matemáticos.

1. Shebang y configuración inicial

#!/bin/bash
# Script para cálculos geométricos

#! indica qué intérprete usar. Para Bash siempre es #!/bin/bash

2. Crear función principal

calcular_geometria() {
    while true; do
        echo "--- Menú ---"
        echo "1. Opción"
        read -p "Seleccione: " opcion
    done
}

3. Estructura case (switch en Bash)

case $opcion in
    1)
        read -p "Ingrese lado: " lado
        area=$(echo "$lado * $lado" | bc -l)
        echo "Área: $area"
        ;;
    2)
        read -p "Ingrese radio: " radio
        pi=3.14159
        area=$(echo "$pi * $radio^2" | bc -l)
        ;;
    *)
        echo "Opción no válida"
        ;;
esac

4. Uso de bc para matemáticas

# bc -l carga la biblioteca matemática estándar
resultado=$(echo "scale=2; 5 * 3.14159" | bc -l)
# scale define decimales
volumen=$(echo "scale=4; (4/3) * $pi * ($radio^3)" | bc -l)

5. Ejecutar la función

# Llamar a la función principal
calcular_geometria

# Al final del script, se ejecuta automáticamente

💡 Consejos Bash

  • Usa bc -l para operaciones con decimales y potencias
  • Las variables se usan con $nombre al leer, pero sin $ al asignar
  • read -p muestra un prompt antes de leer
  • echo -e permite usar caracteres especiales como \n

🐚 Script en Bash

#!/bin/bash

# Función para calcular figuras
calcular_geometria() {
    while true; do
        echo -e "\n--- Calculadora de Áreas y Volúmenes (SENA) ---"
        echo "1. Cuadrado (Área y Perímetro)"
        echo "2. Círculo (Área y Perímetro)"
        echo "3. Cubo (Volumen)"
        echo "4. Esfera (Volumen)"
        echo "5. Salir"
        read -p "Seleccione una opción: " opcion

        case $opcion in
            1)
                read -p "Ingrese el lado del cuadrado: " lado
                area=$(echo "$lado * $lado" | bc -l)
                perimetro=$(echo "4 * $lado" | bc -l)
                echo "Resultado -> Área: $area | Perímetro: $perimetro"
                ;;
            2)
                read -p "Ingrese el radio del círculo: " radio
                pi=3.14159
                area=$(echo "$pi * ($radio ^ 2)" | bc -l)
                perimetro=$(echo "2 * $pi * $radio" | bc -l)
                echo "Resultado -> Área: $area | Perímetro: $perimetro"
                ;;
            3)
                read -p "Ingrese el lado del cubo: " lado
                volumen=$(echo "$lado ^ 3" | bc -l)
                echo "Resultado -> Volumen del Cubo: $volumen"
                ;;
            4)
                read -p "Ingrese el radio de la esfera: " radio
                pi=3.14159
                volumen=$(echo "(4/3) * $pi * ($radio ^ 3)" | bc -l)
                echo "Resultado -> Volumen de la Esfera: $volumen"
                ;;
            5)
                echo "Saliendo del programa..."
                exit 0
                ;;
            *)
                echo "Opción no válida, intente de nuevo."
                ;;
        esac
    done
}

# Ejecutar la función
calcular_geometria

🐍 Script en Python

import math

def calcular_figuras():
    while True:
        print("\n--- Calculadora de Áreas y Volúmenes ---")
        print("1. Cuadrado (Área y Perímetro)")
        print("2. Círculo (Área y Perímetro)")
        print("3. Cubo (Volumen)")
        print("4. Esfera (Volumen)")
        print("5. Salir")
        
        opcion = input("Seleccione una opción: ")

        if opcion == "1":
            lado = float(input("Ingrese el lado del cuadrado: "))
            print(f"Área: {lado**2} | Perímetro: {4*lado}")
        
        elif opcion == "2":
            radio = float(input("Ingrese el radio del círculo: "))
            print(f"Área: {math.pi * radio**2:.2f} | Perímetro: {2 * math.pi * radio:.2f}")
            
        elif opcion == "3":
            lado = float(input("Ingrese el lado del cubo: "))
            print(f"Volumen: {lado**3}")
            
        elif opcion == "4":
            radio = float(input("Ingrese el radio de la esfera: "))
            volumen = (4/3) * math.pi * (radio**3)
            print(f"Volumen: {volumen:.2f}")
            
        elif opcion == "5":
            print("Saliendo del programa...")
            break
        else:
            print("Opción no válida.")

if __name__ == "__main__":
    calcular_figuras()
